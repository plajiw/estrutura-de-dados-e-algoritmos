# dsa-practice

This repository is a compilation of **Data Structures and Algorithms (DSA)** exercises solved across multiple competitive programming and learning platforms.

The goal of this repository is to document consistent practice and problem-solving activities, focusing on fundamental computer science concepts such as algorithms, data structures, and computational thinking.

## Scope

The repository contains solutions and experiments based on problems from platforms including, but not limited to:

- LeetCode  
- Beecrowd  
- GeeksforGeeks  
- HackerRank  
- CSES Problem Set
